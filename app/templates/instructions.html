About this Tool

The geocoding tool in the Geocode Data tab uses online resources (Google maps, Geonames, Open Street Map) to find probable latitude/longitudes for addresses and place names. The Vetting tool in the Vet Results tab takes in the geocoding outputs and plots them on a map, location by location, and allows the user to select the coordinate believed to be the most likely match. Use the drop down menu at the top of the map to select a location, and use the mark best buttons below the map to select the location to mark best in the output.


Inputs to the Geocoding Tool

Infile

Select the button to choose a file to upload to the geocoding tool. The file must be a csv with two columns. The first column (Default: Address) contains place names or addresses to match. The second column (Default: iso2) contains iso2 codes for the suspected country.

Encoding

The encoding of the csv file passed to upload. The default "detect" option will try utf-8 and latin1. Note that addresses with nonlatin characters (i.e. cyrillic) must be in utf-8 or another encoding that supports them. Incorrect encoding specifications may cause strange bugs.

Address Field

The name of the address column in the infile.

ISO2 Field

The name of the iso2 column in the infile.

Geocoding Services and Keys

Use the checkboxes to select which of the three services you'd like to use to geomatch to. Google maps requires an API key, Instructions for which can be googled. Note that you must enter payment information in order for the key to work. Each team that does geocoding should have their API key, and attach their own payment information. Obtain UW credit card info and add it to your cloud account; it is not advised to add a personal credit card. You are allowed 40,000 free queries per month (each row counts as a query), so you have to do quite a bit of geocoding in order to incur a charge. Geonames also requires a valid username, which can be created on the geonames website. This is free and quick to set up.

Results Per Source

The number of results returned from each of the geocoding services. Note that services will only return as many results as can be found in that country.

Buffer Size

The geocoding tool will automatically assign a "best" coordinate if all points fall within a circular buffer of this size (in kilometers).


Inputs to the Vetting Tool

Infile

Select the button to choose a file to upload to the geocoding tool. The file must be a csv outputted from the geocoding tool.

Address Field

The name of the address column in the infile.

ISO2 Field

The name of the iso2 column in the infile.

Encoding

The encoding of the csv file passed to upload. This must be the same as the encoding used to produce the infile in the geocoding tool. Strange things may happen otherwise.
